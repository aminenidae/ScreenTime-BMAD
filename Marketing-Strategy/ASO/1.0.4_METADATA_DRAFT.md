**No-redundancy principle:** Tokens already in the Name (`tic`, `lock`, `parental`, `control`, `app`) and Subtitle (`kids`, `lock`, `app`, `reward`, `chart`) are excluded from the KW field. The ASO community reports that repeating tokens across these fields wastes character budget and can negatively impact ranking — Apple already indexes Name and Subtitle at higher weight, so duplicating those tokens in the KW field adds no new search surface and may dilute signal.

**Exception:** `kids` appears in both Subtitle and KW — kept for `kids mode` phrase formation.

**Strategy:** Freed budget invested in the educational/school niche and device targeting:
- `homework`, `school`, `math`, `education`, `reading` — positions Tic Lock alongside educational search intent
- `ipad` — device-specific searches ("parental control ipad", "screen time ipad")
- `games` — parent searches ("kids games", "screen time games")
- `screen`, `time` — core category terms not in Name/Subtitle after rebrand

**Current EN_US token pool (all fields combined):**
Name `tic, lock, parental, control, app` + Subtitle `kids, lock, app, reward, chart` + KW `parenting, kids, mode, games, child, family, ipad, chore, screen, time, homework, school, math, education, reading`.

**Unique tokens: 23** (deduped across Name + Subtitle + KW).

**History:**

## 6. Name (per locale, IDENTICAL across all 5) — 🔁 REBRANDED 2026-05-23

```
Tic Lock: Parental Control App
```

**30/30 chars.** Rebrand cut 2026-05-23 (Brain Coinz → Tic Lock). Prior values: `Brain Coinz: Parental Control` (Apr 20 lock, shipped in withdrawn build 7), `Brain Coinz: Earn Screen Time` (pre-Apr 20).

The "Parental Control" anchor and Apr-20 rationale below are preserved — only the brand prefix changed and "App" was added as a third indexable token. `Tic Lock` brings the `lock` token into the highest-weight field, reinforcing `screen lock`, `app lock`, and `kids lock` combos that previously only fired off KW + Subtitle.

### Rationale (verified against Astro + competitor scan)

- `parental control` — Pop **41** / Diff 65 (direct phrase landing in Name, highest-weight field)
- `parental control app` — Pop **57** / Diff 65 (combines with Subtitle/KW `app` token)
- `control` — Pop 20 (token unlock for `time control` Pop 20, `screen time control` Pop 50)
- **Category-fit signal:** Competitor scan of top parental-control apps found 15/15 include kid/parental/family signal in Name. `Earn Screen Time` placed Brain Coinz in the "adult self-control" cohort (Opal, one sec, etc.) — wrong search-intent pool. `Parental Control` fixes this.
- **Trade-off accepted:** `limit screen time` Pop 23 / Diff 55 was the pure-ASO alternative; rejected because it reads like an adult self-limiter and loses the parental-market signal entirely.

Do not localize. Brand + category phrase must be consistent globally.
