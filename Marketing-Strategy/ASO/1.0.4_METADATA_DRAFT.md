## ⚠️ Critical Context — Primary Language Discovery

Brain Coinz's **Primary Language = English (Canada)**. The 97-char keyword string currently live (the one we drafted for 1.0.3) sits in the **EN_CA** slot, NOT an EN_US slot. US App Store searchers currently get that metadata only via fallback from primary.

**1.0.4 fixes this** by adding English (U.S.) as a dedicated localization. EN_CA remains primary (that's fine — can't change without deleting the app). US rankings will now index against EN_US metadata directly.

## 6. Name (per locale, IDENTICAL across all 5) — ✅ LOCKED

```
Brain Coinz: Parental Control
```

**29/30 chars.** Changed 2026-04-20 from `Brain Coinz: Earn Screen Time`.

### Rationale (verified against Astro + competitor scan)

- `parental control` — Pop **41** / Diff 65 (direct phrase landing in Name, highest-weight field)
- `parental control app` — Pop **57** / Diff 65 (combines with Subtitle/KW `app` token)
- `control` — Pop 20 (token unlock for `time control` Pop 20, `screen time control` Pop 50)
- **Category-fit signal:** Competitor scan of top parental-control apps found 15/15 include kid/parental/family signal in Name. `Earn Screen Time` placed Brain Coinz in the "adult self-control" cohort (Opal, one sec, etc.) — wrong search-intent pool. `Parental Control` fixes this.
- **Trade-off accepted:** `limit screen time` Pop 23 / Diff 55 was the pure-ASO alternative; rejected because it reads like an adult self-limiter and loses the parental-market signal entirely.

Do not localize. Brand + category phrase must be consistent globally.

## 7. Subtitle (per locale, IDENTICAL across all 5) — ✅ LOCKED

**FINAL (30/30):**

```
Limit screen time, differently
```

### Rationale — hybrid ASO + conversion

- **ASO side (`limit screen time`):** Pop **23** / Diff 55 — the highest-Pop multi-token phrase under Diff 60 that fits a 30-char Subtitle. All 3 tokens (`limit`, `screen`, `time`) now sit in Subtitle (higher-weighted than KW field), giving the phrase stronger indexing signal than the prior split Subtitle+KW layout.
- **Conversion side (`differently`):** Preserves the curiosity-gap tag from the prior rev. Parent reads a familiar category opener then hits "differently" — pattern-break forces the "how?" click.
- **Free bonus combos from the Subtitle trio:** `screen time limit` Pop 11, `time limit` Pop 7, `screen limit` Pop 6, `app limit` Pop 7 (with `apps` in KW). Four winnable phrases from one 30-char slot.
- **Apple 2.3.7 safe** — descriptive, no superlative.
- **Cascade effect on KW field:** `limit` removed from KW (now redundant); freed budget allowed adding `blocker` + `apps` + `points` (see §1 rev 2).
